Passo 3:

Tratando Interação do Usuário

Para permitir aos usuários interagir com o aplicativo, podemos usar a diretiva v-on para anexar escutas a eventos (event listeners) que invocam métodos em nossas instâncias Vue: 

<div id="app-5">
    <p>{{ message }}</p>
    <button v-on:click="reverseMessage">Inverter Mensagem</button>
</div>
var app5 = new Vue({
        el: '#app-5',
        data: {
            message: 'Olá Vue!'
        },
        methods: {
            reverseMessage: function () {
            this.message = this.message.split('').reverse().join('')
        }
    }
})

{{ message }}

Você pode observar que neste método, nós atualizamos o estado da aplicação sem ao menos tocar no DOM - pois uma vez que todas as manipulações são tratadas pelo Vue, o código que você escreve é focado somente na lógica da manipulação dos dados.

Além disso, você Vue também é capaz de prover diretiva v-model, que torna a interligação de mão dupla (two-way binding) entre a caixa de texto e o estado da aplicação simplesmente uma moleza: 

<div id="app-6">
    <p>{{ message }}</p>
    <input v-model="message">
</div>
var app6 = new Vue({
    el: '#app-6',
    data: {
        message: 'Olá Vue!'
    }
})

{{ message }}

Deste modo, você está mais que preparado par seguir para o próximo passo.
